# Digital SAT Classroom Platform

## Deployment Status
- **Last Updated:** 2025-12-16
- **Status:** Ready for Vercel Deployment

## Overview
This platform provides a comprehensive solution for managing Digital SAT preparation:
- **Teachers**: Manage classes, content, and view analytics.
- **Students**: Take tests, track progress, and view performance.
- **Admin**: Manage users and global content.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Database**: Supabase
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run local server: `npm run dev`.
